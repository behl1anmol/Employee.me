# [Employee.me](https://github.com/behl1anmol/Employee.me)
Employee.me is an app to fetch Employee data from a SQL Database by consuming a .NET 6 Web API.

### Platforms

- [x] Android
- [x] Windows

## Features
* Using MS SQL 
* Consuming .NET 6 Web API
* ColletionView
* Activity Indicator
* MVVM
